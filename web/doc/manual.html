Skating Diagram Creator

Overview

To build a skating diagram, enter move codes into the text entry box. Each move should be on a new line, or separated from the previous move by a semi-colon ;

Blank lines are ignored, and anything after a hash # on a line is treated as a comment and ignored.

Move codes are typically made up of:

  • An optional prefix, indicating how a change of foot is performed. For example:
    • a wd- prefix indicates a wide step
    • a xf- prefix indicates a cross in front
    • a xb- prefix indicates a cross behind
  • An edge code, indicating the skating foot, direction, and edge. For example, LBO indicates skating on the left foot, travelling backwards on an outside edge.
  • An optional suffix indicating a move, turn or foot change. For example, LFO3 indicates a three-turn from a left forward outside edge, transitioning to a left backward inside edge.
  • Further optional suffices that adjust parameters for the move. These can take two forms:
    • A collection of comma-separated name=value parameters enclosed in square brackets. For example, LFO [curve=120,len=900] indicates a left forward outside edge that turns 120° and travels 900 cm.
    • Short code markers that adjust the most common parameters. For example, LFO>>++ indicates a left forward edge that curves more and travels further than the default.

Move Reference

Edge

Curving edge

Parameters

  • angle: Angle of rotation from start to finish, in degrees (default [object])
  • len: Length in centimetres (default [object])

Straight edge

Straight edge

Parameters

  • len: Length in centimetres (default [object])
  • label: Replacement label, used if non-empty (default [object])
  • style: Style of line (default [object])

Warp

Move skater to new location/position

Parameters

  • x: Horizontal coordinate to warp to (default [object])
  • y: Vertical coordinate to warp to (default [object])
  • dir: Direction to be facing after warp, in degrees (0 is facing down) (default [object])
  • code: Foot code to start with after warp (default [object])

Rink

Rink depiction

Parameters

  • width: Rink width in centimetres (default [object])
  • length: Rink length in centimetres (default [object])
  • centre-line: Whether to show the centre line (default [object])
  • centre-circle: Size of the centre circle in centimetre, 0 to omit (default [object])
  • centre-faceoff: Whether to show the centre face-off (default [object])
  • mid-lines: Location of mid-lines in centimetres from the centre line; 0 to omit (default [object])
  • goal-lines: Location of goal lines in centimetres from the ends; 0 to omit (default [object])
  • goals: Whether to show the goals (default [object])
  • faceoffs: Whether to show the face-offs (default [object])

Info

Set diagram rendering information

Parameters

  • markers: Whether to show begin/end move markers (default [object])
  • bounds: Whether to show overall bounds (default [object])
  • grid: Grid size to display, 0 for no grid (default [object])
  • margin-x: Horizontal margin (default [object])
  • margin-y: Vertical margin (default [object])
  • move-bounds: Whether to show bounds of each move (default [object])
  • font-size: Font size for labels; 0 for auto-scaling (default [object])
  • stroke-width: Stroke width; 0 for auto-scaling (default [object])

Title

Diagram title

Parameters

  • text: Text of title (default [object])
  • x: Horizontal location of title; -1 indicates automatic centering (default [object])
  • y: Vertical location of title (default [object])
  • font-size: Font size for title; 0 for auto-scaling (default [object])

Label

Diagram label

Parameters

  • text: Text to display (default [object])
  • x: Horizontal location of text (default [object])
  • y: Vertical location of text (increasing down) (default [object])
  • font-size: Font size for label; 0 for auto-scaling (default [object])
  • rotate: Angle to rotate text by (default [object])
