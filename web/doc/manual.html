Skating Diagram Creator

Overview

To build a skating diagram, enter move codes into the text entry box. Each move should be on a new line, or separated from the previous move by a semi-colon ;

Blank lines are ignored, and anything after a hash # on a line is treated as a comment and ignored.

Move codes are typically made up of:

  • An optional prefix, indicating how a change of foot is performed. For example:
    • a wd- prefix indicates a wide step
    • a xf- prefix indicates a cross in front
    • a xb- prefix indicates a cross behind
  • An edge code, indicating the skating foot, direction, and edge. For example, LBO indicates skating on the left foot, travelling backwards on an outside edge.
  • An optional suffix indicating a move, turn or foot change. For example, LFO3 indicates a three-turn from a left forward outside edge, transitioning to a left backward inside edge.
  • Further optional suffices that adjust parameters for the move. These can take two forms:
    • A collection of comma-separated name=value parameters enclosed in square brackets. For example, LFO [curve=120,len=900] indicates a left forward outside edge that turns 120° and travels 900 cm.
    • Short code markers that adjust the most common parameters. For example, LFO>>++ indicates a left forward edge that curves more and travels further than the default.

Move Reference

Edge

Curving edge

Parameters

  • angle: Angle of rotation from start to finish, in degrees (default 20). Can also be modified by a < or > (repeated) suffix.
  • len: Length in centimetres (default 450). Can also be modified by a + or - (repeated) suffix.
  • label: Replacement label, used if non-empty
  • style: Style of line

StraightEdge

Straight edge

Parameters

  • len: Length in centimetres (default 450). Can also be modified by a + or - (repeated) suffix.
  • label: Replacement label, used if non-empty
  • style: Style of line

CoE

Change of edge

Parameters

  • angle: Angle of rotation for each curved part, in degrees (default 90). Can also be modified by a < or > (repeated) suffix.
  • len: Length of each curved part in centimetres (default 450). Can also be modified by a + or - (repeated) suffix.
  • flat_len: Length between edges in centimetres (default 50).
  • delta_angle: Difference in angle for second curved part, in degrees
  • delta_len: Difference in length for second curved part, in centimetres
  • style: Style of line

Warp

Move skater to new location/position

Parameters

  • x: Horizontal coordinate to warp to
  • y: Vertical coordinate to warp to
  • dir: Direction to be facing after warp, in degrees (0 is facing down)
  • code: Foot code to start with after warp

Shift

Move skater relative to current location and direction

Parameters

  • fwd: Distance to shift forward in direction of travel
  • side: Distance to shift sidewards relative to direction of travel
  • rotate: Rotation (clockwise) to perform, in degrees
  • code: Foot code to start with after shift

Rink

Rink depiction

Parameters

  • width: Rink width in centimetres (default 3000).
  • length: Rink length in centimetres (default 6100).
  • centre-line: Whether to show the centre line (default true).
  • centre-circle: Size of the centre circle in centimetre, 0 to omit (default 400).
  • centre-faceoff: Whether to show the centre face-off (default true).
  • mid-lines: Location of mid-lines in centimetres from the centre line; 0 to omit (default 800).
  • goal-lines: Location of goal lines in centimetres from the ends; 0 to omit
  • goals: Whether to show the goals
  • faceoffs: Whether to show the face-offs

Info

Set diagram rendering information

Parameters

  • markers: Whether to show begin/end move markers
  • bounds: Whether to show overall bounds
  • grid: Grid size to display, 0 for no grid
  • margin-x: Horizontal margin (default 50).
  • margin-y: Vertical margin (default 50).
  • move-bounds: Whether to show bounds of each move
  • font-size: Font size for labels; 0 for auto-scaling
  • stroke-width: Stroke width; 0 for auto-scaling

Title

Diagram title

Parameters

  • text: Text of title
  • x: Horizontal location of title; -1 indicates automatic centering (default -1).
  • y: Vertical location of title (default 100).
  • font-size: Font size for title; 0 for auto-scaling

Text

Diagram text

Parameters

  • text: Text to display
  • x: Horizontal location of text (default 100).
  • y: Vertical location of text (increasing down) (default 100).
  • font-size: Font size for label; 0 for auto-scaling
  • rotate: Angle to rotate text by

Label

Diagram label relative to current position

Parameters

  • text: Label to display
  • fwd: Location of text relative to current position, in current direction
  • side: Location of text relative to current position, sideways from current direction
  • font-size: Font size for label; 0 for auto-scaling
  • rotate: Angle to rotate text by
